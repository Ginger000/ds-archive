Detailed Description

An implementation of Dijkstra's, modified to find the shortest path between two points. Utilizes min priority queue (Implemented as heap) to achieve O(E log(V)) complexity. Note that better runtime (O(E+VlogV)) is attainable via ues of a fibonacci heap.
