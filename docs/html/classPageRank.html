void PageRank::calculateRank ( double  alpha,
double  tol 
)
private

Calculates pagerank of all pages. Page Rank is proportional to the posibility of a user browsing the wikipage via links in the page that stays at a particular webpage. Iteratively, each page "votes" its rank to the pages that it has links pointing to, with a correction mimicing random browsing.

Parameters
alphaThe dumping factor, representing the possibility for a user to keep browsing or open a new page
tolTolerance for convergence.
