\doxysubsection{Detailed Description}
A base class for traversal algorithms on wikipedia graphs.

\mbox{\hyperlink{classBFS}{BFS}}, \mbox{\hyperlink{classDFS}{DFS}} will inherit from this class

A derived class provides a traversal by returning instances of \mbox{\hyperlink{classTraversal_1_1Iterator}{Traversal\+::\+Iterator}}.
