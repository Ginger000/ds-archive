\doxysubsection{Detailed Description}
Implementation Notes \mbox{\hyperlink{classGraph}{Graph}} implemented as Adjacency List Map used to keep track of weighted connection vectors PRO\+:
\begin{DoxyItemize}
\item Should use O($\vert$\+V$\vert$+$\vert$\+E$\vert$) space, vs. O($\vert$\+V$\vert$$^\wedge$2) in the alternate adjacency matrix method.
\item Easy to add to graph CON\+:
\item Checking adjacency between vertices is not efficient (worst case O($\vert$\+V$\vert$)) 
\end{DoxyItemize}
